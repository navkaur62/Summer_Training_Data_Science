# Project Overview

## Project Name

Petrol Price Trends Across Indian Cities

## Objective

The objective of this project is to analyze petrol price trends across Indian cities using Power BI.

The dashboard provides:

- Fuel price trends
- Inflation analysis
- Exchange rate impact
- State-wise comparison
- City-wise comparison
- Forecasting
- KPIs

## Tools Used

- Power BI
- Excel
- DAX
- Power Query

## Skills Demonstrated

- Data Cleaning
- Data Modeling
- Dashboard Design
- KPI Creation
- Time Intelligence
- Forecasting
