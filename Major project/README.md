# ⛽ Petrol Price Trends Across Indian Cities

## Project Overview

This Power BI dashboard analyzes petrol price trends across major Indian cities.

The project provides insights into:

- Petrol price fluctuations
- Inflation trends
- Exchange rate impact
- Monthly and yearly analysis
- Forecasting future prices
- State-wise comparison
- City-wise comparison

## Dataset

Dataset contains:

- City
- State
- Petrol Price
- Diesel Price
- Inflation Rate
- Exchange Rate
- Date
- Price Change %
- Population
- Fuel Consumption

---

## Key Insights

- Petrol prices increased steadily over time.
- Exchange rate depreciation impacted fuel prices.
- Metro cities recorded higher average prices.
- Inflation showed a positive relationship with fuel price increases.
