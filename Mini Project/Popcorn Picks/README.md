# 🎬 Popcorn Picks: Movie Database Analysis

## Overview

Popcorn Picks is a comprehensive Movie Database Analysis project that uncovers insights into movie popularity, audience ratings, genres, languages, and release trends. Using Python-based data analysis and visualization techniques, this project transforms raw movie data into meaningful insights through Exploratory Data Analysis (EDA).

The project leverages powerful libraries such as NumPy, Pandas, Matplotlib, and Seaborn to clean, analyze, and visualize data, helping identify patterns and trends within the movie industry.

---

## Objectives

* Clean and preprocess the movie dataset for accurate analysis.
* Perform Exploratory Data Analysis (EDA) to uncover hidden patterns.
* Analyze movie popularity, ratings, and audience engagement.
* Explore genre and language distributions across movies.
* Visualize trends using informative charts and graphs.
* Generate actionable insights from the dataset.

## Dataset Features

The dataset contains key information about movies, including:

* **Title** – Name of the movie
* **Genre** – Movie category
* **Popularity** – Popularity score of the movie
* **Vote Count** – Total number of audience votes
* **Vote Average** – Average audience rating
* **Original Language** – Language in which the movie was originally released
* **Release Date** – Official release date of the movie

## Key Findings

* Movies with higher vote counts tend to achieve greater popularity, indicating strong audience engagement.
* English is the most dominant language in the dataset, reflecting its significant presence in the global film industry.
* A small number of genres account for a large proportion of movie releases.
* Popularity and vote count exhibit a positive correlation, suggesting that widely discussed movies attract more audience interaction.
* Movie production and releases have shown noticeable growth in recent years.
* Most movie ratings are concentrated within a moderate range, indicating relatively consistent audience preferences.
* A few highly popular movies act as outliers, significantly exceeding average popularity levels.
