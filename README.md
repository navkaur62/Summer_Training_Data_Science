# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/navkaur62/Summer_Training_Data_Science.git
```

Move into the project directory

```bash
cd Summer_Training_Data_Science
```

Install dependencies

```bash
pip install -r requirements.txt
```

Open notebooks using:

- Jupyter Notebook
- Jupyter Lab
- VS Code

Power BI dashboards can be viewed using **Microsoft Power BI Desktop**.
